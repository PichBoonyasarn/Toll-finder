1　起動・設定確認

  • フォルダ C:\Users\pichb\work\Coding\toll-finder で npm start を実行する
  • ターミナルに 「toll-finder running at http://localhost:3000」 と表示される
  • ブラウザで http://localhost:3000 を開く — ページが正常に読み込まれる
  • F12 DevTools → Console タブ にエラーが出ていないことを確認
  • 出発地フィールドに会社住所（長岡京）が自動入力されている

2　ルート検索（基本動作）

高速道路あり — 目的地: 東京駅

  • 地図にルートの青い線が表示される
  • 高速 入口IC に名前が表示される（例: 大山崎IC・長岡京IC）
  • 高速 出口IC に名前が表示される（例: 東京IC・首都高出口）
  • 距離 に妥当な数値が表示される（目安: 450〜550 km）
  • 所要時間 に妥当な時間が表示される（目安: 4〜5 時間）
  • ETC料金 に ¥ 金額がオレンジ色で表示される
  • 経由 に高速道路名のチップが表示される（名神・新東名など）

高速道路なし — 出発地: 長岡京市　目的地: 向日市

  • ETC料金: 「高速道路なし（無料）」と表示される
  • 入口IC / 出口IC: 「検出できませんでした」と表示される

📍 位置情報ボタン

  • 📍 をクリック → ブラウザが位置情報の許可を求める
  • 許可後、フィールドに現在地の住所が自動入力される

エラー処理

  • 目的地に「aaaaaaaaa」などの無効な住所を入力 → 赤いエラーメッセージが表示される（クラッシュしない）
  • 出発地を空にして検索 → 「出発地を入力してください」のエラーが表示される

3　ドキュメントスキャン — ファイルアップロード

ドロップゾーン UI

  • 目的地フィールドの下にドロップゾーンが表示されている
  • ファイルをドラッグしてゾーン上に乗せると枠が緑色に変わる
  • ドラッグしてゾーンから離すと枠が元に戻る
  • 「ファイルを選択」リンクをクリック → ファイルピッカーが開く（PDF / DOCX / TXT 対応）
  • ドロップゾーン全体をクリック → ファイルピッカーが開く

座標フォーマット別テスト（.txt ファイルを作成してアップロード）

以下の各フォーマットをテキストファイルに書き込んで1件ずつテストするか、全行を1ファイルに入れてまとめてテストする。
テスト内容 フォーマット 合否
緯度: 34.9153, 経度: 135.6865 ラベル付き小数
北緯35度41分22秒 東経139度41分30秒 日本語 DMS
35°41'22"N 139°41'30"E 英語 DMS
35.6580N, 139.7016E 方位付き小数
34.6785, 135.8324 通常の小数ペア
３５．６８９５，１３９．６９１７（全角数字） 全角→半角変換（NFKC）

ファイル形式テスト

  • .txt ファイルをドロップ → 座標が検出される
  • .docx ファイルをドロップ → 座標が検出される（実際の現場資料推奨）
  • .pdf ファイルをドロップ → 座標が検出される（実際の現場資料推奨）

結果表示

  • アップロード後に座標ピッカーが表示される（ファイル名 ＋ 件数）
  • 各カードに 緯度・経度 と 前後の文脈テキスト が表示される
  • 座標が含まれないファイルをアップロード → 「座標が見つかりませんでした」と表示される

4　座標選択 → ルート検索（統合テスト）最重要

座標を含むファイルをアップロードしてから以下の順番でテストする。
  • 座標カードをクリック → 目的地フィールドに「35.xxxxx, 139.xxxxx」が入力される
  • クリックしたカードに緑のハイライトが付く
  • ルート検索をクリック → ルートが正常に計算される（ジオコーディングをスキップして直接座標を使用）
  • 地図と結果カード（IC名・距離・時間・ETC料金）が正常に表示される

座標クリア確認

  • 座標選択後、目的地フィールドを手動で編集 → 選択座標がリセットされる（次の検索はジオコーディングで実行）
  • 別の座標カードをクリック → 以前のハイライトが消えて新しいカードがハイライトされる

5　エッジケース

  • 2つ目のファイルをアップロード → ピッカーが新しいファイルの結果に更新される（前の結果は消える）
  • 同じ座標が2回含まれるファイルをアップロード → 重複排除されて1件だけ表示される
  • 10件以上の座標が含まれるファイルをアップロード → 最大10件まで表示される
